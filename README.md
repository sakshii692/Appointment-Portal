# Appointment-Portal
This project facilitates appointment booking by patients and improves interaction between doctors and patients.

This project includes;

1.Patient Module
- Patient Registration & Login
- Book Doctor Appointments
- View Appointment History
- Access Prescriptions & Medical Records
- Update Profile Information
  
2. Doctor Module
- Doctor Registration & Login
- Manage Patient Records
- Approve/Reject Appointments
- Add Prescriptions & Notes
- View Scheduled Appointments
  
3. Admin Module
- Manage Doctors & Patients
- Monitor System Activities
- Manage Appointments
- Dashboard & Reports

4. Tech Stack;
-> Frontend
   - HTML
   - CSS
   - JavaScript
   - Bootstrap
-> Backend
   - Java
   - Spring Boot
   - REST API
-> Database
   - MySQL
-> Tools & Platforms
   - Git & GitHub
   - IntelliJ IDEA / Eclipse
   - Postman
   - Maven

5. System Architecture
- Frontend sends requests to backend APIs.
- Spring Boot handles business logic.
- MySQL stores patient, doctor, and appointment data.
- Authentication ensures secure access.
